# misc-hardware

Collection of small hardware projects and KiCad designs developed as experimental ones

## Projects

- [mllse-m2-air-breakout-board](./mllse-m2-air-breakout-board/)
  - Breakout board and modification guide for the MLLSE M2 Air mini PC. Provides two full-size M.2 Key-B SATA slots and 
one M.2 Key-M PCIe x1 slot, includes KiCad project, BOM, gerbers, and step-by-step modification instructions.

- [radxa_a5e_m2_to_rasp_pcie](./radxa_a5e_m2_to_rasp_pcie/)
  - M.2 Key-M to Raspberry Pi PCIe adapter for the Radxa A5E SBC. Allows using Raspberry Pi PCIe HATs via a 
Raspberry Pi PCIe FPC cable.

## How to use

Each project contains its own `readme.md` with detailed information, KiCad project files, and media. 
Open the subproject folder and read its `readme.md` for design details, BOM, and assembly or modification instructions.
